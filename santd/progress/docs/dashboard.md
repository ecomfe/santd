<cn>
#### 仪表盘
通过设置 `type=dashboard`，可以很方便地实现仪表盘样式的进度条。
</cn>

```html
<template>
    <div>
        <s-progress type="dashboard" percent="{{75}}"/>
    </div>
</template>
<script>
import progress from 'santd/progress';

export default {
    components: {
        's-progress': progress
    }
}
</script>
```
