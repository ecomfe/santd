## API

| 参数        | 说明                                           | 类型    | 默认值  |
| ---         | ---                                            | ---     | ---     |
| description | 自定义描述内容                                 | string\|SanNode | - |
| imageStyle  | 图片样式                                       | string  | -       |
| image       | 设置显示图片，为 string 时表示自定义图片地址。 | string\|SanNode         | `Empty.PRESENTED_IMAGE_DEFAULT` |
